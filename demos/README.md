# JTaro Module Demo

## 运行示例

首页确保已安装git和nodejs(6或以上)

```bash
git clone https://github.com/chjtx/JTaro-Module.git

cd JTaro-Module

node src/server.js
```

在浏览器打开`http://localhost:3000/demos/index.html`

## Rollup打包

```bash
# 安装rollup、rollup-plugin-jtaro-module
npm install

node build.js
```

将`build/index.html`拖到浏览器访问
